-----------------------------------------------------------------------------------------------------------------------------------------------------------------
                                                            How to use Alignment Tool
-----------------------------------------------------------------------------------------------------------------------------------------------------------------

The alignment tool performs global and local alignment of a pair of sequences.
On the MENU we can select the type of alignment, pick examples to understand better how the tool works and at the end start performing one type of alignemnt.


To run the alignment tool open the terminal in this directory and write:

                    python3 runMe.py


warning: 
    To write an option whenever is asked, don't use " " or ' ', it is not necessary!
    Make sure your sequence file have fasta format, like the example file in the folder: "protein_sequences.fas"
    Make sure your substitution matrix is similar to the one in the folder: "blosum62.mat"


Features included in this tool:
    - Menu where you can select wich type of alignment you prefer to perform
    - Examples to see how the tool works
    - Files for the examples: Seqs.fas is a short file and protein_sequences is a bigger file.
